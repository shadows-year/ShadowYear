# SHADOWYEAR

> 记录那些在强光下被忽略的转瞬即逝。

![License](https://img.shields.io/github/license/username/shadowyear?color=black&style=flat-square)

### 核心特性
- **光影追踪**: 基于原生 JS 的动态背景光斑交互。
- **极简主义排版**: 结合 Inter 与 Cormorant Garamond 字体美学。
- **响应式展示**: 针对视觉作品优化的非对称网格布局。

### 快速开始
1. `git clone https://github.com/your-username/shadow-year.git`
2. 直接在浏览器打开 `index.html`。
